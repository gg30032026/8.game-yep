# Hướng dẫn Chạy Game "Đua Ngựa Bựa"

Game đã được tối ưu để chạy trực tiếp từ file mà không cần cài đặt server.

## Cách chạy
1. Vào thư mục `games/race/`.
2. Double click vào file `index.html`.
3. Trình duyệt sẽ mở ra và bạn có thể chơi ngay!

## Các tính năng
- **Chế độ Chaos**: Đánh nhau, ném chuối, biến hình xe máy Exciter.
- **Giao diện**: 2D phẳng, nét, dễ nhìn.
- **Điều khiển**: Space hoặc Click để dùng vật phẩm.

## Lưu ý
- Nếu muốn chỉnh sửa code, hãy sửa file `script.js` (file này chứa toàn bộ logic game).
- Các file trong thư mục `src/` là mã nguồn gốc được tách module (dùng để phát triển sau này nếu cần tool build), hiện tại game CHỈ DÙNG `script.js`.
